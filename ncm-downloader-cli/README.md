# ncm-downloader-cli

某个用来听歌的小工具 (´・ω・`)

## 这是什么

一个基于某音乐 API 的命令行小助手～

可以搜搜歌、看看词、存存喜欢的音乐到本地，方便离线的时候听 ♪(´▽｀)

## 安装和使用

需要 Python 3.10+，使用 [uv](https://github.com/astral-sh/uv) 管理依赖。

先确保后端服务跑起来了，然后：

```bash
uv run main.py
```

uv 会自动补全依赖并运行。

## 功能一览

- 搜索（曲/人/碟/单）
- 查看歌词
- 保存到本地，支持选择品质
- 批量操作
- 排行榜 / 每日推荐 / FM
- 一些小自定义设置（命名格式、保存路径等）

注：当前不具备元数据刮削功能，下载完成的歌曲应当手动使用 MusicTag 等工具进行元数据匹配。

## 配置

首次运行会引导你填写后端地址，之后所有设置自动保存在本地。
